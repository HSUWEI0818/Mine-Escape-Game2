README
This package contains the JavaScript for the Mines game and a minimal index.html that references it.

Files:
- game.js    : main JavaScript code for the game (the file you requested)
- index.html : simple HTML file that loads game.js so you can run it locally
- README     : this file

How to use:
1. Unzip the archive.
2. Open index.html in your browser (double-click or drag to the browser) to play.
3. To deploy to Cloudflare Pages, put both files at the repo root and deploy as a static site.
